 (require "html-math.rkt")

This library provides utilities for displaying math formulas in Scribble documents using MathJax 3.

1 SetupðŸ”—

Before using the functions provided in this library, you need to call math-setup once in your document. This will include the MathJax 3 library from a CDN.

procedure

(math-setup) → void?

Includes the MathJax 3 library from jsdelivr.net. This function should be called once at the beginning of your document.

2 Inline MathðŸ”—

procedure

(math-inline tex) → element?

  tex : string?
Creates an inline math element. The tex argument should be a string containing the LaTeX code for the math formula.

For example,
#lang scribble/base
 
@math-inline{E=mc^2} is a famous equation.
produces

\( E=mc^2 \) is a famous equation.

3 Display MathðŸ”—

procedure

(math-display tex) → element?

  tex : string?
Creates a display math element. The tex argument should be a string containing the LaTeX code for the math formula.

For example,
#lang scribble/base
 
@math-display{\sum_{i=1}^n i = \frac{n(n+1)}{2}}
produces

\[ \sum_{i=1}^n i = \frac{n(n+1)}{2} \]

4 Aligned MathðŸ”—

procedure

(math-align lines ...) → element?

  lines : string?
Creates an aligned math environment. Each argument is a line of LaTeX code, which will be aligned at the & character.

For example,
#lang scribble/base
 
@math-align{f(x) &= (x+1)^2
                 &= x^2 + 2x + 1}
produces

\begin{align} f(x) &= (x+1)^2\\&= x^2 + 2x + 1 \end{align}
